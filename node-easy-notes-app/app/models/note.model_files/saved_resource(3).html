                                                                                                                                      Rajeev Kumar Singh Mod Pavan 2 months ago

                                                                                                                                      Hi Pavan,

                                                                                                                                      There wasn't a return statement in the controllers in case of errors. Let's take exports.create method for example. In case of errors, I was responding with 400 error like this -

                                                                                                                                      exports.create = function(req, res) {
                                                                                                                                          if(!req.body.content) {
                                                                                                                                              res.status(400).send({message: "Note can not be empty"});
                                                                                                                                          }
                                                                                                                                          // rest of the code
                                                                                                                                      }

                                                                                                                                      Since there was no return statement, the rest of the code was also getting executed even if we had already written the response.

                                                                                                                                      I've included a return keyword now -

                                                                                                                                      exports.create = function(req, res) {
                                                                                                                                          if(!req.body.content) {
                                                                                                                                              return res.status(400).send({message: "Note can not be empty"});
                                                                                                                                          }
                                                                                                                                          // rest of the code
                                                                                                                                      }

                                                                                                                                      I've made the changes in all the methods where the return keyword was missing. The above error shouldn't come after doing these changes.

                                                                                                                                                Rajeev Kumar Singh Mod Vikas Rokkham 3 months ago

                                                                                                                                                Vikas Rokkham When you define a model in mongoose. It automatically looks for a collection with the plural version of your model name. For example, if your model name is Note, it will look for a collection named notes in the database.

                                                                                                                                                Mongoose creates the collection if it doesn't exist when you insert something in that collection for the first time.

                                                                                                                                                You can also specify a custom name for the collection by defining the model like this -

                                                                                                                                                mongoose.model('Note', NoteSchema, 'mynotes');

                                                                                                                                                Check out Mongoose documentation for more information.

                                                                                                                                                      Rajeev Kumar Singh Mod Brian 4 months ago

                                                                                                                                                      Hi Brian,

                                                                                                                                                      You can use Mongoose Queries to filter Notes using any criteria. Following are few examples -

                                                                                                                                                      Find Notes with a specific title: 

                                                                                                                                                      Note.find({'title': 'My First Note'}, function(err, notes){
                                                                                                                                                          if(err) {
                                                                                                                                                              res.status(500).send({message: "Some error occurred while retrieving notes."});
                                                                                                                                                          } else {
                                                                                                                                                              res.send(notes);
                                                                                                                                                          }
                                                                                                                                                      });

                                                                                                                                                      Find Notes created after a specific Date

                                                                                                                                                      Note.find({'createdAt': {'$gt': new Date(2018, 0, 1) }}, function(err, notes){
                                                                                                                                                          console.log(notes);
                                                                                                                                                          if(err) {
                                                                                                                                                              res.status(500).send({message: "Some error occurred while retrieving notes."});
                                                                                                                                                          } else {
                                                                                                                                                              res.send(notes);
                                                                                                                                                          }
                                                                                                                                                      });

                                                                                                                                                      Find Notes whose content has the word "Test" (Case insensitive).

                                                                                                                                                      Note.find({'content' : { '$regex' : /Test/i } }, function(err, notes){
                                                                                                                                                          if(err) {
                                                                                                                                                              res.status(500).send({message: "Some error occurred while retrieving notes."});
                                                                                                                                                          } else {
                                                                                                                                                              res.send(notes);
                                                                                                                                                          }
                                                                                                                                                      });

                                                                                                                                                      You can refer Mongoose Documentation for more examples.

                                                                                                                                                      Cheers!
